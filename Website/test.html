Move the top left point to the provided coordinates.

Element.scrollTo(x, y);

Move the top left point of the element by the horizontal and vertical values provided.

Element.scrollBy(x, y);

Alternate syntax for arguments for Element.scrollTo() and Element.scrollBy(). Not supported in IE, Edge, or Safari.

const options={
        top: "100px",
        left: "100px",
        behaviour: "smooth" 
      }

behaviour defaults to "auto" but can be set to "smooth".

Element.scrollIntoView();

Element.scrollIntoView(true);

The Boolean indicates whether or not to align top of element to top of the screen.

Options object version not supported in IE, Edge, or Safari.

Element.scrollIntoView({
        behaviour: "smooth" 
        block: "start",
        inline: "nearest"
      });

behaviour defaults to "auto" but can be set to "smooth".

Block defines vertical alignment. Value can be "start", "nearest", "center", or "end"

inline defines horizontal alignment. Value can be "start", "nearest", "center", or "end"
